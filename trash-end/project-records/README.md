# Project records

This directory preserves how the Essays tree was built and how its editorial decisions changed. It supports the works but is not itself part of their reading sequence.

## Index

1. [Project history](project-history.md) — chronological development of the works and their branches.
2. [Decision log](decision-log.md) — adopted decisions, rejected alternatives, and unresolved choices.
3. [Conversation sources](conversation-sources.md) — public-safe provenance for private development material.
4. [Repository architecture audit](repository/audit.md) — the original flat-tree audit and later status notes.
5. [Repository migration report](repository/migration-report.md) — implementation history from the additive layout to Essays → Trash-End → works.

## Boundary

Raw conversation exports, private recordings, and complete machine transcripts are not stored here. Public provenance uses hashes, message identifiers, source descriptions, and distilled decisions.

Earlier versions of individual works belong inside each work's own `legacy/` directory. Early independent texts from the *Archivio di Transizione* remain separate fragments. Project records are neither legacy versions nor fragments.
